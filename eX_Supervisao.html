Legenda
Cores — o status de cada peça do sistema: verde = no ar / funcionando roxo = em construção / em decisão amarelo = futuro (por vir) cinza = plataforma / base
Linhas — → cheia = compõe / administra · ⇢ pontilhada = influência (uma peça alimenta/depende de outra)
Passe o mouse em qualquer bloco com ponto roxo para ler o estado real. Botão ❔ Ajuda abre a explicação completa.

Cores das caixas = status (o quadradinho no canto troca)

No ar / funcionando · Em construção / em decisão · Futuro (por vir) · Plataforma / base (o que sustenta tudo).

Tipos de linha (clique na linha para trocar)

Seta cheia → = compõe / administra: a peça faz parte do sistema e é administrada a partir do centro.
Pontilhada ⇢ = influência: uma peça alimenta ou depende de outra (ex.: Obras ⇢ Estoque = materiais por tarefa; Clientes ⇢ Obras = a raiz comercial precede a obra). Pode dar nome à linha pra registrar o porquê.

Ferramentas

Organizar volta ao layout da supervisão · Grade alinha certinho · Tela cheia dá mais espaço · Desfazer (Ctrl+Z) volta a última ação · Exportar gera o texto (com as notas) pra você me mandar.

Notas de estado

Clique num bloco e em ✎ nota para registrar em que ponto aquela peça está. A nota não fica na tela — aparece só quando você passa o mouse por cima (um pontinho roxo no canto indica que há nota). As notas viajam junto no Exportar.
